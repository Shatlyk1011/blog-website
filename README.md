# SET-WEB


Сайт для публикации постов о веб разработке. Редактор [tiptap](https://tiptap.dev). Для дизайна функционала идеи брал с [dev.to](https://dev.to)
### Доступный функционал
+ Редактор + нструменты: **bold**, _italic_, `code / codeblock`, highlight, h1-h6, link, image, quote и другие 
+ Добавление постов + редакция/удаление (при условии что вы владелец).
+ Возможность просмотра поста как превью.
+ Auto save - сохранение постов как черновик. При желании, можно вернуться и продолжить.
+ Подсчет времени для чтении
+ Общедоступность - навигация по `tab`, aria-label
+ Route Guard для не авторизованных
+ Добавление коментарий под каждый пост
+ Удаление/изменение/лайк - коментарии
+ Авторизация на базе firebase authentication через логин и пароль, либо через google account.
+ update 16.05 поиск по тегам/поиск по названию
